Documentation

Introduction

This api extracts information from data.

Usage

Api uses HTTP, method: PUT, path: /, body: form-data.

Api returns result as JSON object with keys taken from submitted form keys and corresponding values as content descriptions.

Supported forms of data

  • Image formats:
    • PNG
    • JPEG
  • Sound recording formats:
    • MP3
  • Plain text

Recognizable contents in data

  • For images:
    • Faces
    • Letter/Digit recognition
  • For sound recordings:
    • Music
  • For plain texts:
    • Language detection
